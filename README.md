# HaxxToyBox

这是一个基于 BepinEx 为 `il2cpp` 游戏《大侠立志传》开发的 Mod。它展示了如何加载自己的资源到游戏中，并实现实例化。

## 构建和开发

要从源代码构建此 Mod，请确保您已安装以下工具：

- Visual Studio 2019 或更高版本
- .NET 6 SDK

构建步骤：

1. 克隆仓库到本地。
2. 使用 Visual Studio 打开 `HaxxToyBox.sln`。
